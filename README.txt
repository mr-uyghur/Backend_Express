# Backend_Express
a simple backend server codebase with CRUD operation, built with Express and Node.js

This code base and its folder structure is meant to be comprehensive and simple. 

Be sure to download all the dependencies inside package.json before running the server, 
and type in node server.js cmd on terminal inside the directory to run server.

Codebase overview:
Data base is inside the model foler. standard User schema.

Routes with API endpoint are inside routes folder.

controllers for backend logic is inside controllers folder.
